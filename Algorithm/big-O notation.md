# 빅오 표기법 (big-O notation)

## 빅오 표기법
빅오 표기법은 알고리즘의 효율성의 척도를 나타내는 표기법이며 시간 복잡도를 나타내는 방법이다.
데이터의 개수 (n)이 주어졌을 때 연산 횟수를 의미한다.


## 빅오 표기법의 특징
- 상수항 무시
    - 빅오 표기법은 데이터의 입력값이 충분히 크다고 가정하고, 알고리즘의 효울성 또는 데이터의 입력 값의 크기에 따라 영향을 받기 때문에 상수항 같은 사소한 부분은 무시한다.
- 영향력 없는 항 무시
    - 빅오 표기법은 데이터 입력값의 크기에 따라 영향을 받기 때문에 가장 영향력이 큰 항 이외의 나머지 영향력이 없는 항은 무시한다.

## 빅오 표기법 표기 예제
- O(1) : 상수 시간
    - Array에서 append(), Stack에서 Push, Pop등
- O(log n) : 로그 시간
    - binary tree, 2의 제곱근식 등
- O(n) : 선형 시간
    - for문, 선형 탐색 등
- O(n log n) : 로그선형 시간
    - 퀵 정렬, 병합 정렬 등
- O^2 : 2차 시간
    - 이중 for문
